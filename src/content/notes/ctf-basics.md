# CTF Fundamentals

## Getting Started with CTFs

Capture The Flag (CTF) competitions are a great way to develop practical cybersecurity skills.

## CTF Categories

### Web Exploitation

Common challenges:
- SQL injection
- XSS (Cross-Site Scripting)
- CSRF (Cross-Site Request Forgery)
- Server-Side Request Forgery (SSRF)
- Local/Remote File Inclusion

### Cryptography

Topics covered:
- Classical ciphers (Caesar, Vigenere)
- Modern encryption (RSA, AES)
- Hash functions
- Encoding schemes (Base64, Hex)

### Binary Exploitation

Techniques:
- Buffer overflows
- Return-oriented programming (ROP)
- Format string vulnerabilities
- Stack canaries bypass

### Reverse Engineering

Tools and concepts:
- Ghidra
- IDA Pro
- Assembly language basics
- Debugging with GDB

### Forensics

Skills needed:
- Disk imaging and analysis
- Memory forensics (Volatility)
- Network traffic analysis
- Steganography

## Essential Tools

### Reconnaissance
- nmap
- nikto
- dirb/gobuster

### Exploitation
- Burp Suite
- sqlmap
- Metasploit

### Analysis
- Wireshark
- binwalk
- strings

### Misc
- CyberChef (data encoding/decoding)
- John the Ripper (password cracking)
- Hashcat

## Tips for Success

1. **Read the challenge carefully** - Details matter
2. **Start with easy challenges** - Build confidence
3. **Take notes** - Document your findings
4. **Learn from writeups** - Study solutions after completion
5. **Practice regularly** - Consistency is key
